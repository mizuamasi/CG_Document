CG Intro for Blender Class

Blender実習を支えるCG基礎リーディング

このページは「読み物」と「体感デモ」を一体化した教材です。各章で1つだけ操作し、何が変わるかを確認します。

Markdown版 Smooth Shadeへ Normal Mapへ Roughnessへ Lightingへ

0. 最初に押さえること

Blenderで触っている要素は、ほぼ 形（Mesh）・表面（Material）・光（Light） の3つです。デモでは毎回「何を変えたか」「見え方のどこが変わったか」の2点だけ見てください。

1. 頂点・辺・面

モデリングは点・線・面の再配置です。操作名を暗記する前に、いま増えているのが頂点か面かを意識すると理解が安定します。

2. Smooth Shade と法線

Smooth Shadeは形状を増やす機能ではなく、法線を補間して陰影を滑らかに見せる機能です。FlatとSmoothを切り替えて比較します。

3. Normal Map

Normal Mapは法線情報をテクスチャで追加し、凹凸感を増やします。シルエットは同じまま、陰影だけ変わる点を確認します。

4. PBR（Roughness / Metallic）

Roughnessは反射の広がりを決める値です。低いとシャープ、高いとぼけます。Metallicは非金属/金属の反射モデル切り替えです。

5. ライティングと有名な配置

Point / Sun / Areaで影の硬さと方向が変わります。3-Pointにすると、主光・補助光・縁取り光の役割分担が見えやすくなります。

6. まとめ

  • 形の土台は頂点・辺・面
  • Smooth Shadeは法線補間で、形状追加ではない
  • Normal Mapは陰影を増やす技術で、輪郭は変えない
  • Roughnessは反射のぼけ方、Metallicは反射モデルの切り替え
  • ライトは数より役割（Key / Fill / Rim）で考える
