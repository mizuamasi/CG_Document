## GitHub Pages設定

1. GitHubリポジトリの `Settings` を開く
2. `Pages` を開く
3. `Build and deployment` の `Source` を `Deploy from a branch` に設定
4. Branch を `main`、Folder を `/docs` に設定
5. 保存後、表示されたURL（`https://<user>.github.io/<repo>/`）にアクセス
